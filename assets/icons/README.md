# App Icons

## Icon Specifications

### Main App Icon (`app_icon.png`)
- Size: 1024x1024 pixels
- Format: PNG
- Background: Transparent or white
- Design: Professional compass/compass rose icon representing guidance and direction

### Adaptive Icon Foreground (`app_icon_foreground.png`)
- Size: 1024x1024 pixels
- Format: PNG
- Background: Transparent
- Safe area: Keep important content within the center 432x432 pixels (for Android adaptive icons)

## Design Concept

The icon should represent:
- **Compass/Boussole**: Navigation and guidance
- **Student**: Educational context
- **Direction**: Finding the right path
- **University**: Academic context

## Color Scheme

- Primary: Purple/Violet (#6B46C1)
- Secondary: Blue (#3B82F6)
- Accent: Gold/Yellow (#F59E0B)

## Notes

- Use `flutter pub run flutter_launcher_icons` to generate all platform icons
- The adaptive icon background is set to #6B46C1 (purple) in pubspec.yaml
